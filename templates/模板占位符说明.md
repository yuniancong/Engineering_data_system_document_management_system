# Word模板占位符说明

本文档说明各个Word模板文件中需要使用的占位符，以便系统正确填充数据。

## 1. 表三、卷内目录.docx

**模板类型**：表格模板

**说明**：该模板使用表格结构，系统会自动复制模板行并填充数据。

**表格结构要求**：
- 前3行：标题和表头（保留不变）
- 第4行：数据模板行（会被复制）

**列顺序**（从左到右）：
1. 序号
2. 文件材料题名
3. 原字编号
4. 编制单位
5. 编制日期
6. 页次
7. 备注

## 2. 表四、卷内备考表.docx

**模板类型**：占位符模板

**占位符列表**：
```
{{totalPages}}     - 本案卷共有文件材料（页数）
{{textPages}}      - 文字材料（页数）
{{drawingPages}}   - 图样材料（页数）
{{photoCount}}     - 照片（张数）
{{note}}           - 说明
{{creator}}        - 立卷人
{{createDate}}     - 立卷日期
{{reviewer}}       - 审核人
{{reviewDate}}     - 审核日期
```

**使用方法**：
在Word模板的相应位置直接输入占位符（包括双花括号），例如：
```
本案卷共有文件材料 {{totalPages}} 页
立卷人：{{creator}}    日期：{{createDate}}
```

## 3. 表二、档案封面.docx

**模板类型**：占位符模板

**占位符列表**：
```
{{archiveNo}}          - 档号
{{title}}              - 案卷题名
{{unit}}               - 编制单位
{{startDate}}          - 起始日期
{{endDate}}            - 结束日期
{{secretLevel}}        - 密级
{{retentionPeriod}}    - 保管期限
{{totalVolumes}}       - 本工程共（卷）
{{volumeNumber}}       - 本案卷为第（卷）
```

**使用方法**：
```
档号：{{archiveNo}}
案卷题名：{{title}}
编制单位：{{unit}}
起止日期：{{startDate}} 至 {{endDate}}
密级：{{secretLevel}}
保管期限：{{retentionPeriod}}
本工程共：{{totalVolumes}} 卷
本案卷为第：{{volumeNumber}} 卷
```

## 4. 表一、档案移交目录.docx

**模板类型**：表格模板

**说明**：该模板使用表格结构，系统会自动复制模板行并填充数据。

**表格结构要求**：
- 前2行：标题和表头（保留不变）
- 第3行：数据模板行（会被复制）

**列顺序**（从左到右）：
1. 案卷号
2. 案卷题名
3. 文字(页)
4. 图纸(张)
5. 其他
6. 编制单位
7. 编制日期
8. 保管期限

## 5. 附件3-移交书.docx

**模板类型**：占位符模板

**占位符列表**：
```
{{title}}              - 工程名称（案卷题名）
{{unit}}               - 移交单位（编制单位）
{{date}}               - 移交日期（自动生成当天日期）
{{totalVolumes}}       - 案卷数量
{{totalPages}}         - 文件总页数
{{note}}               - 移交说明
{{archiveNo}}          - 档号
{{retentionPeriod}}    - 保管期限
```

**使用方法**：
```
工程名称：{{title}}
移交单位：{{unit}}
移交日期：{{date}}
案卷数量：{{totalVolumes}} 卷
文件总页数：{{totalPages}} 页
移交说明：{{note}}
```

## 注意事项

### 表格模板注意事项：
1. 不要修改表格的列数和列顺序
2. 可以调整表格样式、字体、边框等格式（这些会被保留）
3. 表头行的文字可以修改
4. 数据模板行的单元格格式会被应用到所有数据行

### 占位符模板注意事项：
1. 占位符必须完整包含双花括号，例如 `{{title}}`
2. 占位符区分大小写，必须严格按照上述列表使用
3. 可以在占位符周围添加任何文字和格式
4. 占位符可以在表格单元格中使用
5. 不要在占位符中间插入其他格式标记

### 通用注意事项：
1. 所有模板文件必须是 `.docx` 格式（不支持 `.doc` 格式）
2. 建议在编辑模板时关闭拼写检查，避免Word自动修改占位符
3. 如果占位符没有被正确替换，请检查是否有拼写错误或格式问题
4. 导出前请确保模板文件位于 `templates/` 目录下

## 测试建议

在实际使用前，建议：
1. 准备测试数据
2. 逐个测试每个模板的导出功能
3. 检查导出的Word文档格式和内容是否正确
4. 根据需要调整模板样式
